วิธีติดตั้ง backend

1. เปิด Google Sheets ใหม่ ตั้งชื่อ "บันทึกรับ-ส่งเอกสาร"

2. ไปที่ Extensions → Apps Script

3. วางโค้ดจากไฟล์ Code.gs แล้วกด Save

4. Deploy → New Deployment → Web App → Execute as: Me, Who has access: Anyone

5. Copy URL มาวางในช่อง Apps Script URL ด้านบน
